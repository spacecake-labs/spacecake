## overview

the tasks panel displays tasks from your active Claude Code session. toggle it from the status bar or right-click the tasks button to change its dock position (bottom by default).

if no Claude Code session is connected, the panel shows "no active claude session".

## task list

each task shows:

| column      | description                               |
| ----------- | ----------------------------------------- |
| task        | task ID                                   |
| title       | task subject                              |
| status      | pending, in progress, or completed        |
| description | optional details (hover to see full text) |
| owner       | optional owner badge                      |

the table supports sorting by clicking column headers.

## filtering

use the status filter buttons in the toolbar to show only tasks with a specific status:

- **pending** — tasks that haven't started
- **in progress** — tasks currently being worked on
- **completed** — finished tasks

multiple filters can be active at the same time. click the reset button to clear all filters.
