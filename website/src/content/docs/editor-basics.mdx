## workspaces

a workspace is a folder you're working in. switch workspaces with `‚åòO` / `ctrl+O` or by clicking the sidebar header.

your home folder (`~/.spacecake`) is always available for notes that aren't tied to a specific project.

## sidebar

the sidebar shows your workspace file tree. toggle it with `‚åòB` / `ctrl+B`.

folders load their contents on demand when you expand them. the current open file is highlighted.

### file operations

right-click a file or folder to access the context menu:

**files:**

- rename
- delete
- revert (if the file has unsaved changes)

**folders:**

- new file
- new folder
- rename
- delete

you can also use the **+** button at the top of the sidebar to create a new file or folder in the workspace root.

renaming selects the filename without the extension, so you can quickly type a new name and press enter. press escape to cancel.

## files and tabs

- **quick open**: press `‚åòP` / `ctrl+P` to search and open files by name
- **new file**: press `‚åòN` / `ctrl+N` to create a new file
- **close tab**: press `‚åòW` / `ctrl+W` or click the √ó on the tab
- **save**: press `‚åòS` / `ctrl+S` to save the current file
- **save all**: press `‚åò‚áßS` / `ctrl+shift+S` to save all open files

tabs show open files at the top of the editor. scroll horizontally if you have many tabs open.

if a file is modified outside spacecake while you have unsaved changes, you'll see a conflict indicator. you can choose to keep your changes or discard them and reload from disk.

## autosave

spacecake can automatically save files as you edit. enable autosave in workspace settings:

1. click the gear icon in the sidebar
2. toggle **enable autosave** on

when enabled, changes are saved to disk automatically.

## themes

toggle between light and dark mode using the theme button in the status bar. the theme persists across sessions.

note: the terminal theme is locked when each tab is created. see [troubleshooting](/troubleshooting#terminal-doesnt-match-the-app-theme) if the terminal doesn't match after switching themes.

## panels

spacecake has three panels ‚Äî [terminal](/terminal), [git](/git), and [tasks](/tasks). each panel can be docked to the left, bottom, or right of the editor.

**default positions:**

| panel    | default position |
| -------- | ---------------- |
| terminal | right            |
| git      | left             |
| tasks    | bottom           |

to change a panel's dock position, right-click its toggle button in the status bar and choose a new position. drag the panel edge to resize it. collapsing a panel preserves its state ‚Äî the terminal keeps running, git status stays loaded.

see the [keyboard shortcuts](/keyboard-shortcuts) page for a complete shortcut reference.
